## mailmint 
this app is a mailing app like **gmail**

**with the features :** 
users can create their mail ids and then send mails to their friends and family but on mailmint only not in other apps.
  
**how can you use this in your device :**
 - clone the repository 
 - go on each folder " cd client " & "cd server" 
 - run on each " bun install "
 - for client run "bun run dev"
 - for server run "bun run dev"
 - and then you go ...
